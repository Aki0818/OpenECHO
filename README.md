OpenECHOは、[ECHONET Lite][]プロトコルをJavaで実装したライブラリです。
現在機器オブジェクトとして分電盤とエアコンがサポートされています。
機器クラスを継承し、自分の機器ノードを作成することもできます。

詳しいドキュメントはもうしばらくお待ちください。
